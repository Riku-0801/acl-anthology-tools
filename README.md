# ACL Anthology Tools

ACL Anthology（aclanthology.org）の使い勝手を向上させるChrome拡張機能。

## 機能一覧

| 機能 | 対象ページ |
|------|-----------|
| BibTeXワンクリックコピー | 論文一覧・詳細ページ |
| アブストラクトインラインプレビュー | 論文一覧ページ |
| PDF直接アクセスボタン | 論文一覧ページ |
| 論文ブックマーク管理 | 論文一覧・詳細ページ |
| リアルタイム検索・絞り込み | イベントページ（`/events/*`） |
| 著者ポップアップ（最近の論文） | 論文一覧ページ |
| 既読・未読トラッキング | 論文一覧・詳細ページ |
| ブックマーク／既読データエクスポート | 拡張機能ポップアップ |

## 開発環境のセットアップ

```bash
npm install
```

## ビルド

```bash
# 開発ビルド（ファイル変更を監視して自動再ビルド）
npm run dev

# 本番ビルド
npm run build
```

ビルド成果物は `dist/` ディレクトリに出力されます。

## Chromeへのインストール

1. `npm run build` を実行して `dist/` を生成する
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`dist/` フォルダを選択する

以後、`npm run dev` を起動した状態でソースを変更すると自動再ビルドされます。拡張機能ページの更新ボタンを押すと変更が反映されます。

## 使い方

### BibTeXコピー

- **一覧ページ**: 論文エントリにカーソルを合わせると「BibTeX」ボタンが表示されます。クリックするとクリップボードにコピーされます。
- **詳細ページ**: ページ上部に常時表示される「BibTeXをコピー」ボタンをクリックします。

### アブストラクトプレビュー

論文タイトルにカーソルを合わせるとアブストラクトがポップアップ表示されます。ポップアップ内からBibTeXコピー・PDF表示・ブックマーク追加が操作できます。`Escape` キーで閉じます。

アブストラクトプレビューのオン/オフは拡張機能ポップアップの「設定」タブから切り替えられます。

### PDFクイックアクセス

論文エントリにカーソルを合わせると「PDF」ボタンが表示されます。クリックすると新しいタブでPDFが開きます。ArXivバージョンが存在する場合は追加オプションとして表示されます。

### ブックマーク

- **追加**: 論文エントリまたは詳細ページのブックマークボタンをクリックします。
- **確認・削除**: 拡張機能アイコンをクリックして開くポップアップにブックマーク一覧が表示されます。
- **エクスポート**: ポップアップの「ブックマークをJSONでエクスポート」ボタンからダウンロードできます。

### リアルタイム検索（イベントページ）

`aclanthology.org/events/` 配下のページでは、論文一覧の上部に検索バーが挿入されます。キーワードを入力するとタイトル・著者名・アブストラクトを対象にリアルタイム絞り込みが行われ、「N件 / 全M件」と件数が表示されます。検索バーを空にすると全件表示に戻ります。

### 著者ポップアップ

論文エントリの著者名にカーソルを合わせると、その著者の最近の論文（最大5件）がポップアップ表示されます。「全著作を見る」リンクから著者ページを新しいタブで開けます。

### 既読・未読トラッキング

- **既読マーク**: 論文エントリの既読ボタンをクリックすると既読状態が保存され、視覚的なインジケーターが表示されます。
- **訪問済み自動記録**: 論文詳細ページを初めて開くと自動的に訪問済みとして記録されます（明示的な既読マークとは区別されます）。
- **フィルタリング**: 一覧ページの「すべて表示 / 未読のみ / 既読のみ」フィルターで表示を切り替えられます。
- **エクスポート**: ポップアップの「既読記録をCSVでエクスポート」ボタンからダウンロードできます（paperId, isRead, markedReadAt, visitedAt 形式）。

## プロジェクト構成

```
src/
├── types/index.ts              # 型定義（PaperEntry, BookmarkEntry など）
├── constants/selectors.ts      # DOMセレクター定数
├── utils/domExtractor.ts       # DOM抽出ユーティリティ
├── services/
│   ├── storage.ts              # chrome.storage.local ラッパー
│   ├── bibtex.ts               # BibTeX取得・クリップボードコピー
│   ├── author.ts               # 著者情報フェッチ・キャッシュ管理
│   └── search.ts               # テキストフィルタリング（純粋関数）
├── background/service-worker.ts # HTTPフェッチ委譲（MV3 Service Worker）
├── content-scripts/
│   ├── events.ts               # /events/* 用（検索バー）
│   ├── listing.ts              # 一覧ページ用（各種ボタン注入）
│   └── detail.ts               # 論文詳細ページ用
└── popup/
    ├── index.html              # 拡張機能ポップアップ UI
    └── index.ts                # ポップアップロジック
```

## 技術スタック

- TypeScript 5.x（strict mode）
- Chrome Extension Manifest V3
- Vite + vite-plugin-web-extension
- Vitest + happy-dom（テスト）
- データ保存: `chrome.storage.local`（最大10MB）
- 外部サービスへの通信なし（aclanthology.org へのフェッチのみ）
